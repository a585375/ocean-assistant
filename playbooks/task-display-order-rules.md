# Task Display Order Rules

## 目标
让 Ocean 查看任务时，顺序稳定、符合直觉。

## 默认排序优先级
1. 今天到期 / 今天要做
2. 有明确时间的任务
3. 高优先级且高风险任务
4. 近期任务
5. 长期任务
6. 后台任务最后展示

## 展示分组建议
- 先展示“今天最该做的”
- 再展示“后续任务”
- 最后展示“后台进行”

## 工作 / 生活拆分
当用户问：
- “我今天还有什么任务” -> 默认混合展示，但先展示最临近和最重要的
- “工作的任务还有哪些” -> 只展示工作相关项目
- “家庭 / 宝宝相关还有什么” -> 只展示对应项目

## 展示样式
- 一个任务一个代码块
- 任务所有信息放在同一个代码块里
- 少 emoji，偏干净、结构化、Claude Code 风格
