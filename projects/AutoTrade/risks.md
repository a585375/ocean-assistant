# AutoTrade Risks

## 当前关键风险

### 1. 回测与模拟盘结果不一致
- 风险等级：高
- 影响：会直接影响策略可信度、参数判断与实盘决策
- 当前状态：待逐笔复盘定位根因

### 2. IB / Gateway 运行环境复杂
- 风险等级：高
- 影响：断连、client_id 冲突、行情订阅恢复失败会影响模拟盘/实盘连续性
- 当前状态：已有较多恢复补丁，但长期升级兼容性仍需关注

### 3. Monkey patch 过多
- 风险等级：中高
- 影响：短期提升稳定性，长期增加维护和升级风险
- 当前状态：已识别为重点审计区域

### 4. 配置与运行环境漂移
- 风险等级：中高
- 影响：回测、模拟盘、实盘之间行为不一致
- 当前状态：需要持续比对数据源、参数、风控和执行逻辑

### 5. 敏感配置暴露风险
- 风险等级：中
- 影响：可能带来安全与运维问题
- 当前状态：此前已发现样例配置疑似包含真实敏感信息，后续需要专项清理
