# AutoTrade Context

## 项目定位
一个接近生产化的量化交易平台，而不只是策略脚本。

## 关键组成
- 策略层：NQ 信号，QQQ / QQQ 期权执行
- 回测层：历史回测与基线验证
- 优化层：参数搜索与实验迭代
- 实盘层：IB 接入、运行、恢复
- 平台层：Web API、Discord 控制、日志与运维支持

## 当前已知事实
- 仓库已 clone 到本地
- 已建立 Python 虚拟环境并安装依赖
- 已对 IB 接入与 monkey patch 做过初步审查
- 已创建“模拟盘 vs 回测差异复盘” checklist

## 当前目标
- 先把回测与模拟盘差异的根因找出来
- 再逐步沉淀关键项目知识、风险、决策与复盘结论
